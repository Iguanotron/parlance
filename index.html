Introduction

The Parlance Language is an open-source, C-based language designed for accessibility. The language is defined to be as close to natural language as possible, in order that it can be easily recognized by speech-to-text applications. It uses an easy-to-learn, natural language scheme focused on readability and ease of parsing. In its current form, Parlance is converted to standard ANSI C code by the C Preprocessor

Comparison to C

C Code:

#include <stdio.h>

int main() 
{
	printf("Hello, World!\n");
	return 0;	
}

Parlance Code:

CPP INCLUDE "<stdio.h>"

DECLARE FUNCTION "main" TYPE "int"	
BEGIN
	CALL "printf"
	PARAM LITERAL STRING "Hello world!" LITERAL END
	CALL END
SCOPE END
