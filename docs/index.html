Parlance - Accessible Programming

Readable. Speakable. Powerful.

Purpose

Right now, there is a severe lack of truly accessible options for programmers with physical disabilities. If you can't type, you basically can't code--but we hope to change that.

Modern and old-fashioned languages alike tend to be very symbol-heavy. Even languages that seem very close to natural language (like Python) often rely on strict formatting rules to function, which isn't feasible for people with screen readers, or anyone who operates computers with speech-recognition tools.

Overview

The Parlance language is an open-source, C-based language designed for accessibility. Parlance's syntax is as close to natural language as possible, so that it can be easily recognized by speech-to-text applications. It uses an easy-to-learn set of dictionary words to build a true, powerful programming language focused on readability and speakability. Parlance is directly converted to standard ANSI C code, so users who know C can easily begin using Parlance.

Current Features

  • Turing complete
  • Unambiguous formatting
  • Compiles directly to C code
  • Easy to understand when read aloud
  • Easy to learn with a foundation in C or its relatives
  • Potential for easy speech-to-text and/or text-to-speech integration

Comparison to C

C Code:

#include <stdio.h>

int main() 
{
	printf("Hello, World!\n");
	return 0;	
}

Equivalent Parlance Code:

CPP INCLUDE <stdio.h>

DECLARE FUNCTION main TYPE int	
BEGIN
	CALL printf
		ARG LITERAL STRING "Hello, World!\n" END LITERAL
	END CALL END STATEMENT
	RETURN LITERAL INT 0 END LITERAL END STATEMENT
END SCOPE
END

FizzBuzz in Parlance:

CPP INCLUDE <stdio.h>

DECLARE FUNCTION main TYPE int
BEGIN
DECLARE VARIABLE i TYPE int END STATEMENT 
LOOP 
	FIRST VARIABLE i GETS LITERAL INT 1 END LITERAL
	CONDITION VARIABLE i LESSTHAN LITERAL INT 101 END LITERAL
	CYCLE PREINCREMENT VARIABLE i
BEGIN
	IF VARIABLE i MODULO LITERAL INT 3 END LITERAL EQUALS LITERAL INT 0 END LITERAL AND 
	   VARIABLE i MODULO LITERAL INT 5 END LITERAL EQUALS LITERAL INT 0 END LITERAL
	BEGIN
       	CALL printf 
       		ARG LITERAL STRING "FizzBuzz" END LITERAL 
       	END CALL END STATEMENT
	END SCOPE

	ELSE IF VARIABLE i MODULO LITERAL INT 3 END LITERAL EQUALS LITERAL INT 0 END LITERAL
	BEGIN
       	CALL printf 
       		ARG LITERAL STRING "Fizz" END LITERAL 
       	END CALL END STATEMENT
	END SCOPE

	ELSE IF VARIABLE i MODULO LITERAL INT 5 END LITERAL EQUALS LITERAL INT 0 END LITERAL
	BEGIN
       	CALL printf 
       		ARG LITERAL STRING "Buzz" END LITERAL 
       	END CALL END STATEMENT
	END SCOPE

	ELSE
	BEGIN
       	CALL printf 
       		ARG LITERAL STRING "%d" END LITERAL ARG VARIABLE i 
       	END CALL END STATEMENT
	END SCOPE

	CALL printf ARG LITERAL STRING "\n" END LITERAL END CALL END STATEMENT
END SCOPE

END SCOPE

END
