Parlance - Accessible Programming

Overview

The Parlance Language is an open-source, C-based language designed for accessibility. The language is defined to be as close to natural language as possible, in order that it can be easily recognized by speech-to-text applications. It uses an easy-to-learn, natural language scheme focused on readability and speakability. In its current form, Parlance is directly converted to standard ANSI C code, so users who know C can easily being using Parlance.

Comparison to C

C Code:

#include <stdio.h>

int main() 
{
	printf("Hello, World!\n");
	return 0;	
}

Equivalent Parlance Code:

CPP INCLUDE <stdio.h>

DECLARE FUNCTION main TYPE int	
BEGIN
	STATEMENT CALL printf
		ARG LITERAL STRING "Hello world!\n" END LITERAL
	END CALL
	STATEMENT RETURN LITERAL INT 0 END LITERAL
END SCOPE
